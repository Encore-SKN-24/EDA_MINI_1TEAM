## 1. 프로젝트 개요
   - 본 프로젝트는 국내 시/도단위 전력 사용량 및 기상 데이터를 활용하여, 기후 요인이 전력수요 변화에 미치는 정량적 영향을 탐색하는 데 목적을 두고 있음
   - 기간 : 2026.01.30 ~ 2025.02.04


## 2. 프로젝트 배경 및 필요성

## 3. EDA & 데이터분석 방향성 
1. **전력 사용량과 온도 데이터의 추세 확인**: 전력 사용량과 평균 온도의 전반적인 추세 파악 
2. **계약종별 전력사용량 확인**: 계약종별(주택·일반·산업용등) 중 가장 큰 전력 소비 주체 파악
3. **상관관계 탐색**: 기온, 기압, 습도 등 기상 변수와 전력량의 변수간 연관 관계 분석
4. **직접적 영향 요인 식별**: 전력 사용량의 비대칭적 분포에 유의미한 영향을 미치는 핵심 날씨 요소 도출

 
## 4. 데이터 전처리 과정 및 전반적인 프로세스
  - 본 프로젝트는 각종 날씨 데이터(기온, 습도, 강수량 등)가 전력 사용량에 대한 관계와 영향을 확인하는 것을 목표로 수행함
    - 전반적인 프로세스 : **공공데이터(날씨, 전력사용량) 전처리 → EDA(선형성 확인) → 상관관계 분석(변수 간 연관 관계 확인) → 일반화선형모형(비선형 날씨 데이터가 전력사용량의 비대칭성에 영향을 미치는 날씨 요인 확인)**
    - 결측치: (지역&연도 그룹 별 평균 값/눈이 안오는 날( NA → 0 대체)) →  *ex) 진도군의 3월 '평균기온(°C)'의 결측치는 '고창'의 3월달 평균)
    - 이상치 및 중복제거: 전국날씨데이터'와 '전력사용량데이터' 모두 이상치, 중복 데이터가 없음을 확인.
    - 통계 분석 : 상관분석, 일반화 선형 모델(Generalized Linear Model, GLM)
    - 시각화 : 선 그래프, 히트맵, 박스플롯

## 5. 활용 데이터의 변수 설명
### 5.1. 날씨 데이터 컬럼 설명

| 컬럼명 | 설명 |
|--------|------|
| 평균기온(°C) | 해당 지점의 하루 평균 기온 |
| 평균현지기압(hPa) | 해당 지점의 하루 평균 기압(지점 기준) |
| 평균해면기압(hPa) | 해당 지점의 하루 평균 해면 기준 기압 |
| 평균수증기압(hPa) | 공기 중 수증기가 주는 압력 |
| 평균상대습도(%) | 공기 중 수분 함량 비율 |
| 월합강수량(00~24h만)(mm) | 하루 합계 강수량을 월 단위로 합산한 값 |
| 평균풍속(m/s) | 하루 평균 풍속 |
| 일조율(%) | 하루 중 햇빛이 비친 시간 비율 |
| 최심적설(cm) | 하루 중 가장 깊은 적설(눈 깊이) |
| 평균지면온도(°C) | 하루 평균 지면 온도 |
| CDD (Cooling Degree Day) | 냉방도일 – 냉방 필요 정도를 나타내는 지표(비선형 변수) |
| HDD (Heating Degree Day) | 난방도일 – 난방 필요 정도를 나타내는 지표(비선형 변수) |

<small>CDD (Cooling Degree Day, 냉방도일) = $\max(T_\text{평균} - 24, 0)$</small>  
<small>HDD (Heating Degree Day, 난방도일) = $\max(18 - T_\text{평균}, 0)$</small>

## 6. 전력 사용량과 온도 데이터의 추세 확인


### 6.1. 대한민국 전체 지역의 연도 별 전력사용량 합계 추이
![전력사용량 추이](./assets/korea_monthly_power_usage_trend.png)

#### 그래프 해석
* **지속적인 상승 추세**: 2015년부터 2024년까지의 데이터를 보면, 붉은색 점선(추세선)이 우상향하고 있어 **전국 전력 사용량이 매년 꾸준히 증가**하고 있음을 알 수 있음
* **계절적 변동성(비대칭성)**: 매년 여름(8월)과 겨울(1월)에 전력 사용량이 급증하는 **V자형 패턴**이 반복됩니다. 특히 여름철 냉방 수요와 겨울철 난방 수요가 전력 피크의 주요 원인으로 가정을 해볼 수 있음
* **역대 최고치 경신**: 2024년 여름, 전력 사용량이 **524.2억 kWh**를 기록하며 관측 기간 중 가장 높은 수치를 보임

---

### 6.2. 대한민국 기온 상승 추세 (평균기온)
![기온 상승 추세](./assets/korea_monthly_temp_trend.png)

#### 그래프 해석
* **뚜렷한 기온 상승**: 검은색 점선(기온 변화 추세선)을 통해 지난 10년간 대한민국의 **평균 기온이 약 1.83°C 상승**했음을 확인할 수 있음
* **여름철 고온 현상 심화**: 2015년 최고 평균기온인 25.0°C에 비해, 2024년에는 **28.0°C**까지 치솟으며 여름이 점점 더 뜨거워지고 있음
* **전력 사용량과의 관계**: '전력사용량 추이' 그래프와 비교했을 때, 최고 기온이 발생하는 시점과 전력 사용량 피크 시점이 일치함  → 이는 **기온 상승이 직접적으로 전력 수요 급증을 유발**한다고 가정해 볼 수 있음
* 10년 사이에 연평균 기온이 크게 상승했는데 이는 24년이 113년 관측 역사상 가장더운 한해로 역대 최고 기록을 14.5°C로 경신했기 때문임
* 출처 : https://www.kma.go.kr/kma/news/press.jsp?mode=view&num=1194448

## 8. 지역별 산업용 전력 소비량 기상 영향 분석 (GLM)

본 프로젝트는 지역별 산업용 전력 소비 데이터와 기상 변수 간의 관계를 **일반화 선형 모델(Generalized Linear Model, GLM)**을 통해 분석한 결과임

---

## 📊 통계 분석 개요 (GLM)
전력 소비량은 기온 등 기상 변수와 냉·난방 수요의 결합으로 인해 비선형적인 패턴(U자형)을 보이며, 본 분석은 다음과 같은 방법론을 적용함

* **Gamma 분포 가정**: 전력 소비량은 0 이상의 연속형 변수이며 비대칭적인 분포(Tail Distribution)를 가지며, 이를 정규분포로 가정하는 선형 회귀보다 **Gamma 회귀 모델**이 산업 공정의 냉각·가열 효율 변화를 분석하는 데 적합함
* **비선형성 고려**: 냉·난방도일(HDD, CDD) 변수를 포함하여 날씨와 전력 간의 복합적인 관계를 모델링함

## 💡 주요 인사이트 및 결과 해석

본 분석을 통해 도출된 지역별 산업 전력 소비의 핵심 특징과 통계적 시사점은 다음과 같음

### 1. 강력한 전력 수요 견인 변수 (Sensitivity Analysis)
기온과 습도는 산업 공정의 핵심 변수로 작동하며, 특정 지역에서 폭발적인 증폭 효과를 보임
* **충청도 평균기온 (exp: 11.06)**: 기온이 1단위 변화할 때 전력 사용량이 **약 11.1배의 강도**로 매우 민감하게 반응함  →  이는 반도체 클린룸 등 외부 온도에 극도로 민감한 **초정밀 항온항습 공정**이 밀집된 지역적 특성이 반영된 결과임
* **경상남도 평균수증기압 (exp: 8.05)**: 수증기압이 1단위 증가할 때 전력 사용량이 **약 8.1배의 강도**로 급증함  → 이는 기온보다 '습도'가 전력 수요를 더 강력하게 견인함을 의미하며, 산업용 공조 시스템이 공기 중 수분을 제거하는 **제습(잠열 제거) 과정**에 막대한 에너지를 소모하고 있음을 입증함

### 2. 계수(coef) 부호에 따른 전력 소비 패턴의 통계적 입증
계수의 부호는 기준점 대비 전력 사용량의 상대적 증감을 나타내는 중요한 지표임
* **음수(-) 계수의 의미**: 강원도와 충청도의 '월(Month)' 변수 계수가 음수로 나타나는 것은, 전력 소비가 극에 달하는 **1월(기준점)** 대비 타 계절의 조업 환경이나 난방 부하가 완화되어 사용량이 줄어드는 패턴을 통계적으로 보여줌
* **지면온도의 반전 현상**: 지면온도가 **반비례(-, exp 1미만)**로 나타나는 충청도(0.137)와 경상남도(0.063)는 겨울철 차가운 지표면으로 인한 **공장 바닥 및 기초부 열 손실 방지**를 위한 히팅 부하가 산업용 전력 소비의 숨은 복병임을 나타냄


---
> **💡 수치 해석 가이드**
> - **계수(coef) (+) / 지수화(exp) > 1**: 해당 기상 수치 상승 시 전력 사용량 **증가**
> - **계수(coef) (-) / 지수화(exp) < 1**: 해당 기상 수치 상승 시 전력 사용량 **감소**
> - **충청도 기온(exp: 11.06)**: 기온 1단위 변화 시 전력 사용량이 약 11배의 강도로 민감하게 반응함(기온 변화가 산업용 냉방 부하에 매우 강력한 증폭 효과를 줌을 시사함)
